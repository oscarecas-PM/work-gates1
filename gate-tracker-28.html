Gate Tracker v1.0

Load the Gate Decoder Ring and Daily Upload CSV files, then click Process.

1. Gate Decoder Ring

Columns: Part No., Part Name, Gate Number, Gate Name, Oper No., Work Dept, Modifier

2. Daily Upload

Columns: Order No, Operation No, Operation title, Operation Status, Part Number, Program, Serial Numbers, Hold Status, ...

3. Master Gates (optional)

Columns: Gate Number, Gate Name, Technician 1, Technician 2, ... Technician 10

4. Mobile Audit CSV (optional)

Columns: Gate Number, Order No., Part No., Serial No., Technician (optional)

5. Audit History Log (optional)

Columns: Date, Gate Number, Order No, Part No, Serial No, Technician

6. P File Export(s) (optional, weekly)

XLSX file(s) with WO, Baseline, Total Actual Hours, Percent Complete. Multiple files OK.

7. VAC Tracker Log (optional)

Cumulative EVM history CSV. Upload previous export to see trends & delta VAC.
